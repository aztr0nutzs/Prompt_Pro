# PROMPT-FORGE

Offline-first prompt library application.

## Run
Open app/index.html in any modern browser.

## Structure
- app/ : Dark GUI web app
- prompts/ : Prompt library


## Theme
- app/css/rsi_theme.css: verbatim from your provided style.css
- app/css/styles.css: PROMPT-FORGE layout + mint/neon classification overrides
